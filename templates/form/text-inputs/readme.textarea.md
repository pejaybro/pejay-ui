# Textarea Component

A styled form input component optimized for Tailwind.

---

## Usage

```tsx
import { Textarea } from "@/pejay-ui/components/form";

<Textarea
  label="Remarks"
  rows={6}
/>
```

---

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `rows` | `number` | `4` | Default display rows. |
