# UrlInput Component

A styled form input component optimized for Tailwind.

---

## Usage

```tsx
import { UrlInput } from "@/pejay-ui/components/form";

<UrlInput
  placeholder="https://example.com"
/>
```

---

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `label` | `string` | `"URL"` | Input field label. |
