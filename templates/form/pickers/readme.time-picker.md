# TimePicker Component

A styled form input component optimized for Tailwind.

---

## Usage

```tsx
import { TimePicker } from "@/pejay-ui/components/form";

<TimePicker
  label="Start Time"
  value={time}
  onChange={setTime}
/>
```

---

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `value` | `string` | — | Time string value (e.g. "12:00 AM"). |
