# Radio Component

A styled form input component optimized for Tailwind.

---

## Usage

```tsx
import { Radio } from "@/pejay-ui/components/form";

<Radio
  label="Option Item"
  name="group-1"
/>
```

---

## Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| `label` | `string` | — | Option description text. |
