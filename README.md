# pejay-ui

A lightweight CLI tool to initialize, add, and remove React UI components in your projects.

## Core Commands

### 1. Initialize Configuration
```bash
npx pejay-ui init
```

### 2. Add Component
```bash
npx pejay-ui add <component-name>
```

### 3. Remove Component
```bash
npx pejay-ui remove <component-name>
```
